SpreeUserGroups
===============

Introduces user groups and calculator to add adjustment for specified user group.


Installation
============

      Add `gem "railscarma_user_groups", github: 'RailsCarma/railscarma_user_groups', branch: '2-1-stable'`
      Run `bundle install`
      Run `rake db:migrate`
